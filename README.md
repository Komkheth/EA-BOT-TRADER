# Scalper + BTC EA Bot

## 🎯 Overview
Complete Forex EA Bot with **Scalper strategy** and **BTC trading** support. Runs 24/7 with MT5 integration.

## ✨ Features
- ⚡ **Scalper Strategy**: Fast trades on EURUSD, GBPUSD, USDJPY
- ₿ **BTC Trader**: Bitcoin/USD trading with technical analysis
- 📊 **Real-time Dashboard**: Live monitoring with WebSocket
- 🔄 **Auto-trading**: Cron-based automated execution
- ⚙️ **Configurable**: Adjust lot sizes, SL, TP, risk
- 📈 **Performance Tracking**: Win rate, profit/loss, daily trades

## 🚀 Quick Start

### Installation
```bash
git clone https://github.com/yourusername/forex-scalper-btc-ea.git
cd forex-scalper-btc-ea
npm install
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your MT5 credentials
npm start
